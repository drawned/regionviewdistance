# Region View Distance
A simple PaperMC plugin that adds two new flags to World Guard:
- view-distance
- simulation-distance

---

# Requires
- Requires PaperMC
- Requires [WorldGuard](https://dev.bukkit.org/projects/worldguard)
- at least Minecraft version 1.17
